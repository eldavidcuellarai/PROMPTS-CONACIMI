# Prompts de IA para Miembros de Cámaras de Comercio (Concamin)

Este documento recopila prompts de Inteligencia Artificial específicamente diseñados para miembros de cámaras de comercio de Concamin, cubriendo generación de imágenes, búsqueda profunda y análisis de documentos con NotebookLM.

## Prompts para Generación de Imágenes

### Marketing Empresarial

1.  Prompt:
    ```
    Genera una imagen de un producto innovador en un entorno de comercio electrónico, con un un diseño limpio y minimalista que resalte sus características. Estilo: fotografía de producto, alta resolución, iluminación de estudio.
    ```

2.  Prompt:
    ```
    Crea una imagen de un equipo de trabajo colaborando en una oficina moderna, que transmita profesionalismo y dinamismo. Estilo: fotografía corporativa, natural y auténtica.
    ```

3.  Prompt:
    ```
    Diseña una imagen para una campaña de redes sociales que promueva un servicio de consultoría empresarial, utilizando elementos gráficos que simbolicen crecimiento y éxito. Estilo: ilustración vectorial, colores corporativos, diseño amigable.
    ```

4.  Prompt:
    ```
    Genera una imagen de un evento de networking empresarial, con personas interactuando, tarjetas de presentación y un ambiente profesional y amigable. Estilo: fotografía de eventos, dinámico y con enfoque en las interacciones.
    ```

5.  Prompt:
    ```
    Crea una imagen de un logotipo abstracto que represente la globalización y el comercio internacional, con líneas fluidas y colores que inspiren confianza. Estilo: diseño de logotipo, moderno y versátil.
    ```

## Prompts para Búsqueda Profunda

### Investigación de Mercados

1.  Prompt:
    ```
    Realiza una búsqueda exhaustiva sobre las tendencias emergentes en el turismo de aventura en México, identificando los destinos más populares, el perfil demográfico de los viajeros y las proyecciones de crecimiento para los próximos 5 años. Incluye datos de fuentes confiables como la Secretaría de Turismo, INEGI y organismos internacionales.
    ```

2.  Prompt:
    ```
    Investiga el impacto de la digitalización en el sector de pequeñas y medianas empresas (PyMES) en Guanajuato. Enfócate en la adopción de comercio electrónico, herramientas de marketing digital y el uso de plataformas de pago en línea. Proporciona ejemplos de casos de éxito y desafíos comunes.
    ```

3.  Prompt:
    ```
    Analiza el mercado de productos orgánicos en México, con un enfoque en la demanda del consumidor, los principales productores y distribuidores, y las oportunidades de exportación para empresas de Guanajuato. Incluye regulaciones y certificaciones relevantes.
    ```

4.  Prompt:
    ```
    Identifica las principales oportunidades de inversión en el sector automotriz de Guanajuato, considerando la llegada de nuevas empresas, la expansión de las existentes y la cadena de suministro local. Proporciona datos sobre la fuerza laboral calificada y los incentivos gubernamentales.
    ```

5.  Prompt:
    ```
    Realiza una investigación sobre las preferencias de los turistas internacionales que visitan Guanajuato. ¿Qué tipo de alojamiento prefieren? ¿Qué actividades realizan? ¿Cuál es su gasto promedio? Utiliza encuestas, estudios de caso y datos de plataformas de reservas.
    ```

### Análisis de Competencia

1.  Prompt:
    ```
    Identifica a los principales competidores de Guanajuato como destino turístico a nivel nacional e internacional. Analiza sus estrategias de marketing, sus ventajas competitivas y sus debilidades. ¿Qué podemos aprender de ellos para mejorar la promoción turística de Guanajuato?
    ```

2.  Prompt:
    ```
    Realiza un análisis de la competencia para una empresa de calzado en Guanajuato que busca expandirse al mercado de exportación. Identifica a los principales fabricantes de calzado en países como Italia, España y Brasil. Compara sus modelos de negocio, precios, calidad y canales de distribución.
    ```

3.  Prompt:
    ```
    Investiga las estrategias de precios de los hoteles boutique en ciudades coloniales de México similares a Guanajuato. ¿Cómo fijan sus tarifas? ¿Qué servicios adicionales ofrecen? ¿Cómo se diferencian de la competencia?
    ```

4.  Prompt:
    ```
    Analiza las campañas de promoción de inversión de otros estados mexicanos que compiten con Guanajuato por la atracción de empresas. ¿Qué incentivos ofrecen? ¿Qué sectores priorizan? ¿Cómo comunican sus ventajas competitivas?
    ```

5.  Prompt:
    ```
    Identifica a los principales proveedores de servicios logísticos en la región del Bajío. Compara sus tarifas, tiempos de entrega, cobertura y especialización. ¿Cuál sería el proveedor más adecuado para una empresa de manufactura en Guanajuato?
    ```

### Tendencias Económicas

1.  Prompt:
    ```
    Investiga las proyecciones de crecimiento económico para México y Guanajuato en los próximos 3-5 años, considerando factores como la inflación, las tasas de interés, el tipo de cambio y el comercio internacional. Utiliza fuentes como el Banco de México, INEGI, FMI y el Banco Mundial.
    ```

2.  Prompt:
    ```
    Analiza el impacto de la inteligencia artificial y la automatización en el mercado laboral de Guanajuato. ¿Qué sectores serán los más afectados? ¿Qué nuevas habilidades serán demandadas? ¿Qué programas de capacitación se están implementando para preparar a la fuerza laboral?
    ```

3.  Prompt:
    ```
    Realiza una búsqueda sobre las tendencias globales en sostenibilidad y economía circular, y cómo estas pueden aplicarse al desarrollo económico de Guanajuato. Identifica oportunidades para la inversión en energías renovables, gestión de residuos y producción sostenible.
    ```

4.  Prompt:
    ```
    Investiga el papel de las cadenas de suministro globales en la economía de Guanajuato, especialmente en el sector automotriz y de manufactura. ¿Cuáles son los riesgos y oportunidades actuales? ¿Cómo se están adaptando las empresas a los cambios geopolíticos y tecnológicos?
    ```

5.  Prompt:
    ```
    Analiza las tendencias en el consumo de bienes y servicios en México, con un enfoque en los cambios en los hábitos de compra, el crecimiento del comercio electrónico y la demanda de productos personalizados. ¿Cómo pueden las empresas de Guanajuato adaptarse a estas tendencias?
    ```

### Oportunidades de Negocio

1.  Prompt:
    ```
    Identifica nichos de mercado no explotados en el sector turístico de Guanajuato, como el turismo de bienestar, el turismo cultural especializado o el turismo rural. Proporciona ejemplos de modelos de negocio exitosos en estas áreas y cómo podrían replicarse en Guanajuato.
    ```

2.  Prompt:
    ```
    Realiza una investigación sobre las oportunidades de negocio en la economía plateada (mercado de adultos mayores) en Guanajuato. ¿Qué productos y servicios son demandados por este segmento de la población? ¿Cómo pueden las empresas adaptar sus ofertas para satisfacer estas necesidades?
    ```

3.  Prompt:
    ```
    Analiza las oportunidades de negocio que surgen de la implementación de la Zona Económica Especial (ZEE) en Guanajuato (si aplica, o un proyecto similar de desarrollo regional). ¿Qué sectores se verán beneficiados? ¿Qué incentivos fiscales y facilidades se ofrecen a las nuevas empresas?
    ```

4.  Prompt:
    ```
    Investiga las oportunidades de negocio en el desarrollo de software y servicios tecnológicos para la industria manufacturera en Guanajuato. ¿Qué tipo de soluciones son más demandadas (ej. Industria 4.0, IoT, Big Data)? ¿Qué programas de apoyo existen para startups tecnológicas?
    ```

5.  Prompt:
    ```
    Identifica oportunidades de negocio en la exportación de productos agroindustriales de Guanajuato a mercados internacionales. ¿Qué productos tienen mayor potencial? ¿Cuáles son los requisitos de importación en los mercados objetivo? ¿Qué apoyos gubernamentales existen para la exportación?
    ```

## Prompts para Análisis de Documentos con NotebookLM

### Reportes Económicos

1.  Prompt:
    ```
    Analiza el reporte adjunto del INEGI sobre el Producto Interno Bruto (PIB) de Guanajuato. Identifica los sectores económicos que mostraron mayor crecimiento y aquellos que experimentaron contracción. Explica las posibles causas de estas variaciones y su impacto en la economía local.
    ```

2.  Prompt:
    ```
    Revisa el informe de la Secretaría de Economía sobre la balanza comercial de Guanajuato. Identifica los principales productos de exportación e importación, así como los países socios comerciales más importantes. Evalúa si existe un superávit o déficit comercial y sus implicaciones.
    ```

3.  Prompt:
    ```
    Examina el reporte de inflación del Banco de México (documento adjunto) y su impacto en el poder adquisitivo de los habitantes de Guanajuato. Identifica los bienes y servicios que han experimentado mayores aumentos de precio y sugiere posibles medidas para mitigar sus efectos.
    ```

4.  Prompt:
    ```
    Analiza el informe de empleo y salarios del IMSS (documento adjunto) para el estado de Guanajuato. Identifica los sectores con mayor generación de empleo y aquellos con mayor crecimiento salarial. Compara estos datos con el promedio nacional y evalúa la calidad del empleo generado.
    ```

5.  Prompt:
    ```
    Revisa el reporte de inversión extranjera directa (IED) en Guanajuato (documento adjunto). Identifica los países de origen de la inversión, los sectores receptores y el monto total de la IED. Evalúa el impacto de esta inversión en la creación de empleo y el desarrollo tecnológico.
    ```

### Estudios de Mercado

1.  Prompt:
    ```
    Analiza el estudio de mercado adjunto sobre las preferencias del consumidor para productos de calzado en México. Identifica las tendencias de diseño, los rangos de precios aceptables y los canales de compra preferidos. Resume las oportunidades para una marca de calzado de Guanajuato.
    ```

2.  Prompt:
    ```
    Revisa el estudio de viabilidad adjunto para la apertura de un nuevo hotel boutique en San Miguel de Allende. Evalúa la demanda proyectada, la competencia existente, los costos de inversión y los ingresos esperados. Determina si el proyecto es financieramente viable y cuáles son los principales riesgos.
    ```

3.  Prompt:
    ```
    Examina el estudio de mercado adjunto sobre la demanda de servicios de logística y transporte en la región del Bajío. Identifica los tipos de servicios más solicitados, los principales clientes y las expectativas de crecimiento. Sugiere estrategias para una empresa de logística que busca expandir su participación en el mercado.
    ```

4.  Prompt:
    ```
    Analiza el estudio de mercado adjunto sobre las oportunidades de exportación para productos agroindustriales de Guanajuato en el mercado estadounidense. Identifica los productos con mayor potencial, los requisitos sanitarios y fitosanitarios, y los canales de distribución. Resume las barreras de entrada y las estrategias para superarlas.
    ```

5.  Prompt:
    ```
    Revisa el estudio de mercado adjunto sobre la adopción de tecnologías de la Industria 4.0 en las empresas manufactureras de Guanajuato. Identifica las tecnologías más implementadas, los beneficios percibidos y los desafíos. Sugiere programas de capacitación o incentivos para fomentar una mayor adopción.
    ```

### Documentos Empresariales

1.  Prompt:
    ```
    Analiza el plan de negocios adjunto de una startup tecnológica. Identifica la propuesta de valor, el modelo de ingresos, el análisis de mercado y las proyecciones financieras. Evalúa la solidez del plan y señala posibles áreas de mejora o riesgos.
    ```

2.  Prompt:
    ```
    Revisa el contrato de asociación adjunto entre dos empresas. Resume las cláusulas clave relacionadas con la distribución de ganancias, las responsabilidades de cada parte y los mecanismos de resolución de disputas. Identifica cualquier ambigüedad o riesgo legal.
    ```

3.  Prompt:
    ```
    Examina el informe de auditoría financiera adjunto de una empresa. Identifica las principales observaciones, las recomendaciones y cualquier irregularidad detectada. Evalúa la salud financiera de la empresa con base en este informe.
    ```

4.  Prompt:
    ```
    Analiza el manual de operaciones adjunto de una planta manufacturera. Resume los procedimientos clave de producción, los estándares de calidad y las medidas de seguridad. Identifica posibles cuellos de botella o áreas donde se podría mejorar la eficiencia.
    ```

5.  Prompt:
    ```
    Revisa el informe de sostenibilidad adjunto de una empresa. Identifica las iniciativas clave en materia ambiental, social y de gobernanza (ESG). Evalúa el compromiso de la empresa con la sostenibilidad y su impacto en la comunidad y el medio ambiente.
    ```
